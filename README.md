# VIP视频解析

一个简洁的VIP视频解析工具，支持一些主流视频平台。

## 下载
- 到 *releases* 里选择版本下载
- 不同版本功能差异不大，主要是*UI*界面更新
- 目前仅支持 *Windows* 平台

## 功能

- 支持多平台解析
- 15条解析线路可选
- 一键解析播放
- 全屏播放支持
- 支持弹幕

## 使用方法

1. 前往 *releases* 下载压缩包
2. 解压下载的压缩包
3. 双击 *.exe* 文件运行
4. 复制视频网址，粘贴到输入框
5. 选择线路，点击「解析播放」

## 注意事项

- 播放异常时请尝试更换线路
- 请勿相信暂停时的广告(毕竟是*免费* 的)
